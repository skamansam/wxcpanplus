Information Pane

This pane is located at the bottom of the Modules Tab. It is where you can get various information about a module. In order to get the desired information, simply single-click on any item in the Modules list. When you click on an item in the Modules list, all information in the Information Pane is reset.

The Information Pane contains the following items:

  • Info Tab
  • Actions Tab
  • Files Tab
  • Readme.txt Tab
  • Status Tab
  • Prerequisites Tab
  • Contents Tab
  • Report Tab
  • Validate Tab
