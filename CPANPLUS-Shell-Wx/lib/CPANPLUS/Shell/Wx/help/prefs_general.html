wxCPAN Preferences :: General

These settings relate to general CPANPLUS Configuration options.

Show Startup Tips
Shows tips when wxCPAN is started.
Verbose output
Outputs a lot of information when using CPANPLUS.
Do Not Update Sources
Updates CPANPLUS's source files
Prefer External Binaries
Whether or not use external binaries instead of modules. (i.e. `zip` instead of Compress::Zlib)
Debug
Show a lot of debug output.
Write Install Log
Write an install log after each install.
Default Shell
The default shell to use when cpanp is executed. Use CPANPLUS::Shell::Wx for wxCPAN. the default is CPANPLUS::Shell::Default
