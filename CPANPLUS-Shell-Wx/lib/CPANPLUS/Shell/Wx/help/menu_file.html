wxCPAN File Menu

The File menu has three items:

  • Open Actions...
  • Save Actions...
  • Quit

Open Actions...

This menu item opens a previously saved batch-process file, which contains all the items in the main Actions tab. This file is simply a batch-process list of modules and actions. After you open the file, click on the 'Run' button in the actions tab to begin processing the actions.

Save Actions...

This menu item simply stores the actions file to disk, which contains all the items in the main Actions tab. This file is simply a batch-process list of modules and actions. After you save the file, you can re-open the file and click on the 'Run' button in the actions tab to begin processing the actions.

Quit

This item quits wxCPAN.
