wxCPAN Preferences :: Build

These settings relate to how CPANPLUS builds the modules.

Allow Interactive Build
Whether to run `perl Makefile.PL` interactively.
Test CPAN
Mail test results to http://testers.cpan.org
Prefer Makefile
Prefer using Makefile.PL instead of Build.PL if both are present.
Skip Tests
Skip testing modules before installing.
Force Install
Files should be forcefully overwritten if tests fail or a newer version already exists. This option can be overridden using the Force option in the Actions Tab.
Flush Cache
Flush cache directories, status flags, etc. after every operation.
Check MD5
Check the MD5 sum for each module after it is downloaded. Only applies if Digest::MD5 is installed.
Check Signature
Check signatures if packages are signed. Only applies if Crypt::OpenPGP is installed.
Use Storable
Use Storable (module) to write package information to disk. This is recommended, as it makes CPANPLUS start up faster.
Extra Build Flags
Any extra flags to be passed to `perl Makefile.PL`. See Module::Build documentation for more information.
Extra Make Flags
Any extra flags to be passed to the make program.
Extra MakeMaker Flags
Any extra flags to be passed to the make program.
Distribution Type
Default distribution type to use when building packages. See perldoc for cpan2dist or CPANPLUS::Dist for more information.
Prerequisite Action
What to do when a module has prerequisites.
Ask: ask whether to install.
Follow: Install prerequisites.
Ignore: Do nothing.
