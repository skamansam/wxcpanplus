wxCPAN Preferences :: Network

This section relates to network connectivity.

Use Passive Mode For FTP
Use passive ftp connections. recommended enabled.
Email
The email address to use for ftp connections and as a sender for emails to testers.cpan.org
Mail Exchange
The mail server to use when sending out emails to testers.cpan.org. If you leave this empty, your system settings will be used.
Network Timeout
The time, in seconds, to allow all network transactions before they time out.
