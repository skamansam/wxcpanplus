wxCPAN Actions Tab

The Actions tab is where you can review a list of actions for later processing. You can add items to this list by right-clicking on a module in the Modules tab and selecting "Batch->[action]" from the menu. You can remove an item from this list by clicking on the Remove button (Clear button to clear all).

If you want to save the list for later processing, select 'Save Actions...' from the File menu. If you want to continue a previously saved session, select 'Open Actions...' from the File menu.
