Searching For Modules

Search Terms

In the main search bar, there are two methods for entering search terms.

Universal

Just type in a search term for a normal, case-insensitive search.

Regular Expression

You can type in a regular expression value by enclosing your search in forward slashes.
For example: "/^wx/i" will find any text that begins with wx and is case-insensitive. For a good tutorial on regular expressions, see http://www.troubleshooters.com/codecorn/littperl/perlreg.htm

Search by Type

In the dropdown menu, there are several types you can search by.

Author

The name of the author.

Comment

The package comment

Cpanid

The CPAN Author ID.

Description

The module description.

Email

The Author's email

Module

The name of the module.

Mtime

The last modified time.

Name

The name of the module.

Package

The package name.

Path

The path of the file, as found on the CPAN server.

Status

The status of the module.

Version

The module version.
