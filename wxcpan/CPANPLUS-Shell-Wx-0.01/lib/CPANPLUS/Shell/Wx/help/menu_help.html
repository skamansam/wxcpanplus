wxCPAN Help Menu

The File menu has four items:

  • Dump Config to Console
  • Show POD Reader
  • Help
  • About

Dump Config to Console

This item dumps the contents of the CPANPLUS configuration to the Log tab, so you can easily examine all the current configuration settings. Mainly used for development.

Show POD Reader

Shows the POD Reader Tab in the main Notebook. See the POD Reader section for more information.

Help

Shows a window containing this help file.

About

Shows a window containing credits, licensing, and general inforamtion regarding wxCPAN.
